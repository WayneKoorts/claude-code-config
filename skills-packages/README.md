# Skills Packages

This directory contains packaged skills (`.skill` files) that can be uploaded to Claude via the web interface at [claude.ai](https://claude.ai).

## Usage

1. Go to [claude.ai](https://claude.ai)
2. Navigate to your profile settings
3. Go to the Capabilities page
3. In the Skills section of the Capabilities page, upload the `.skill` file to add the skill to your account

For more information on skills, see the [Claude Skills documentation](https://docs.anthropic.com/en/docs/claude-code/skills).

## Available Packages

| Package | Description |
|---------|-------------|
| `mdl2-icon-picker.skill` | Icon picker for Segoe MDL2 Assets (WinUI Windows apps) |
| `word-template-generator.skill` | Word document (.docx) generator using OOXML |
